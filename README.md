# ⏰ Digital Clock

A simple and responsive Digital Clock built using HTML, CSS, and JavaScript.

## 📌 Features

* Displays current time in 12-hour format
* Shows AM/PM indicator
* Updates automatically every second
* Responsive and modern user interface
* Beginner-friendly project

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## 📂 Project Structure

```text
Digital-Clock/
│
├── index.html
├── style.css
└── script.js
```

## 🚀 How It Works

1. JavaScript's `Date()` object retrieves the current system time.
2. The hour value is converted from 24-hour format to 12-hour format.
3. AM or PM is determined based on the current hour.
4. `setInterval()` updates the clock every second.
5. The formatted time is displayed on the webpage.
